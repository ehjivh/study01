# Python Study Repository (study01)

本リポジトリは、Pythonプログラミングの基礎習得およびアルゴリズムの検証を目的とした学習用スクリプト集である。

## 概要

Pythonの文法確認、標準ライブラリの利用、および小規模なアプリケーションのプロトタイプ作成を通じて得られたコードを管理している。主に基礎的なデータ処理やロジックのテストを目的としたスクリプトが含まれる。

## ファイル構成

リポジトリ内の主要なスクリプトとその役割は以下の通りである。

* `myapp.py`: 特定の機能（アプリケーション形式のプロトタイプ）を実装したメインスクリプト。
* `test.py` ~ `test5.py`: Pythonの各機能（ループ、条件分岐、リスト操作等）を検証するためのテストコード群。
* `test-2.py` / `test2.py`: 特定のアルゴリズムや外部ライブラリの動作確認用。
* `kokok.py`: 実験的なコード、または特定の計算処理を行うためのスクリプト。
* `.gitignore`: Python実行環境で生成される一時ファイル（`__pycache__`等）を除外するための設定。

## 学習トピック

本リポジトリを通じて、以下の項目について学習・実装を行っている。

* Pythonの基本文法（制御構造、関数定義）
* 標準入出力およびデータ構造（リスト、辞書等）の操作
* デバッグ手法とテストコードの記述
* コーディング規約の遵守とプロジェクト構成の理解

## 実行方法

各スクリプトは標準的なPython 3環境で動作する。

```bash
python test.py

```
